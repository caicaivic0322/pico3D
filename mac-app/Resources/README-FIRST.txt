TrellisMac is a local desktop shell for the trellis-mac repository.

This app bundle does not include:
- Python virtual environments
- Hugging Face model weights
- Generated OBJ/GLB output files
- The cloned TRELLIS.2 upstream checkout

On first launch, choose the trellis-mac repository folder in the app if it is not detected automatically, then run the setup workflow from inside the app.
